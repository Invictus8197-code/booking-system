# Pragya Yog Booking Website

This is a simple yoga booking website built using HTML and CSS. It allows users to view yoga sessions and book a preferred time slot through a form interface.

## Features
- Home page with basic information  
- Booking form with user details and time selection  
- Confirmation page after booking  
- Clean and simple UI  

## Technologies Used
- HTML  
- CSS  

## Project Structure
- index.html  
- booking.html  
- confirmation.html  
- style.css  

## Purpose
This project was created to practice basic web development concepts and build a beginner-level portfolio project.
